## Common Issues and fixes

### Script does not load
Make sure Roblox is fully open before injecting the script executor. Restarting the game often fixes temporary load issues.

### ESP is not visible
Check whether the ESP toggle is enabled and confirm your graphics settings are not interfering with rendering.

### Silent aim feels off
Adjust sensitivity and cursor settings first. Some users search for **mmv script silent aim** builds with custom values, but default settings should always be your starting point.

### Mobile controls are unresponsive
Reopen the executor and rejoin the game. On mobile, stability often depends on the executor version.

### Script runs but features are delayed
This usually means the game has not fully initialized. Wait a few seconds after joining before enabling advanced toggles.

### Crashes after update
A Roblox update may have changed internal behavior. Recheck your executor version and reload the latest script file.

### Confusing UI behavior
Reset the config to default values and test each section separately.
